## 서비스 한 줄 정의

OKR과 하루 업무기록을 연결해, 오늘의 실행이 목표 달성과 어떻게 이어지는지 추적하는 업무회고 서비스.

## 개발 조건

- 프론트엔드: React + TypeScript + Tailwind
- 백엔드/DB/Auth: Supabase
- 배포: Vercel

## MVP 핵심 가치

이 MVP의 핵심은 아래 3개가 실제로 동작하는 것이다.

1. OKR을 만든다.
2. 특정 날짜에 업무기록을 남긴다.
3. 그 기록이 어떤 OKR과 연결되는지 확인한다.

즉, `기록 → 목표 연결 → 현황 확인` 루프를 완성하는 것이 목표다.

## MVP 포함 기능

### 인증

- 회원가입 / 로그인
- 로그아웃
- 보호 라우트

### OKR

- OKR 생성 / 목록 / 수정 / 삭제
- 상태 직접 변경
- 상태 값: `active`, `completed`, `archived`

### 업무기록

- 날짜별 업무기록 작성 / 조회 / 수정 / 삭제
- 하루 1기록
- 작성/수정은 모달 폼
- 한 번에 전체 저장

### TODO

- `todo_items` 분리 저장
- 오전 / 오후 구분
- 여러 개 추가 가능
- 완료 체크 가능
- 삭제 가능

### 업무기록 ↔ OKR 연결

- 업무기록 1개에 여러 OKR 연결 가능
- archived OKR은 연결 대상에서 제외

### 대시보드

- 기록된 일수
- 활성 OKR 개수
- 완료 OKR 개수
- OKR 달성률 목록
- 오늘 할 일
- 내일 할 일 미리보기
