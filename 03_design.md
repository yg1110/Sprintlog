# 디자인 방향

전체 디자인 스타일은 아래 서비스들을 참고한다.

## 참고 스타일:

https://toss.im
https://linear.app
https://notion.so
https://vercel.com

## 디자인 목표:

모던하고 깔끔한 SaaS UI
정보 밀도는 높지만 복잡하지 않게
토스 스타일의 카드 기반 UI
Linear 스타일의 미니멀 인터페이스
Notion 스타일의 여백과 가독성

## 컴포넌트 정책

가능하면 shadcn/ui 컴포넌트를 적극 활용한다.

사용 가능한 주요 컴포넌트 예시
Button
Card
Dialog
Input
Textarea
Badge
Select
Checkbox
Calendar
DropdownMenu
Tooltip
Tabs

단, 프로젝트 요구사항에 맞게 Tailwind로 커스터마이징해도 된다
